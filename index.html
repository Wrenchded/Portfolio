My Projects

KIRAYA - Hustle Free Travel

KIRAYA is a public transportation service designed specifically for Delhi Transport Corporation (DTC) buses. It aims to simplify fare management, enhance the commuter experience, and provide a digital solution for seamless transit. KIRAYA includes features like an e-wallet for easy balance top-ups, KIRAYA cards for convenient fare payments, a real-time vehicle tracking system, and dedicated profiles for passengers and ticket collectors.

AUTOMALL - Premium Automotive Dealership

Automall is a dealership management system tailored for exotic and sports cars, with a focus on managing and facilitating the sale of both new and pre-owned high-performance vehicles. This platform allows users to browse, book, and inquire about cars, offering a premium experience with dedicated pages for luxury brands and models.

TOYOTA - Car Dealership Management System

This dealership management system is tailored specifically for Toyota, aimed at providing a comprehensive platform for users to explore, book test drives, and inquire about new Toyota models. The system prioritizes user convenience and engages potential buyers through features that showcase Toyota’s range, including SUVs, sedans, and hybrids, to meet various customer needs.

FORD - Car Price Prediction System - Machine Learning model using EDA

This project is a machine learning model designed to predict the prices of Ford cars based on various features in the dataset. By applying Exploratory Data Analysis (EDA) and predictive modeling, this system provides accurate pricing insights that can aid in decision-making for both buyers and sellers. The model leverages data visualization and feature engineering techniques to enhance prediction accuracy.
